Kine Fall – Static Website on AWS

About This Project

This static website was deployed using Amazon S3 and Terraform as part of the Cloud Programming course portfolio. It demonstrates how to build, configure, and publish a globally accessible website using Infrastructure as Code (IaC).

The infrastructure was automated with Terraform, making it reproducible and scalable. Public access was carefully configured using bucket policies and website hosting settings.
